
# Дослідження файлів SQLite (.sqlitedb)

Файли з розширенням *.sqlitedb - це бази даних SQLite, які часто зустрічаються в мобільних додатках (особливо iOS) та інших програмах. 

Покроковий підхід до їх експертного дослідження:

## 1. Попередній огляд та фіксація

**Створіть хеш-суми** файлів перед дослідженням (MD5, SHA-1, SHA-256) для підтвердження цілісності доказів.

## 2. Інструменти для дослідження

**Основні інструменти:**
- **DB Browser for SQLite** (SQLiteBrowser) - безкоштовний графічний інтерфейс
- **SQLite Expert** - професійний інструмент з додатковими функціями
- **SQLite командний рядок** - стандартна утиліта sqlite3

**Криміналістичні інструменти:**
- **Autopsy/Sleuth Kit** - включає модулі для SQLite
- **Belkasoft Evidence Center** - аналіз мобільних даних
- **Oxygen Forensic Detective** - для мобільної криміналістики
- **AXIOM** (Magnet Forensics)

## 3. Процес дослідження

### Перевірка цілісності бази
```bash
sqlite3 filename.sqlitedb "PRAGMA integrity_check;"
```

### Отримання структури бази
```sql
.schema  -- переглянути всі таблиці та їх структуру
.tables  -- список таблиць
```

### Аналіз метаданих
- Дата створення/модифікації файлу
- Версія SQLite
- Кодування тексту
- Журнали транзакцій (WAL файли)

### Витяг даних
Експортуйте дані у форматі CSV або JSON для подальшого аналізу та включення до висновку експерта.

## 4. Важливі аспекти для експертизи

**Шукайте:**
- Видалені записи (можливе відновлення через unallocated space)
- Timestamps (часові мітки активності)
- Геолокаційні дані
- Переписку (для месенджерів)
- Історію активності

**Звертайте увагу на:**
- Таблиці з префіксами "deleted_" або "backup_"
- Поля з binary data (BLOB) - можуть містити зображення
- Зв'язки між таблицями (foreign keys)

## 5. Документування

У висновку експерта вкажіть:
- Назву та хеш-суми досліджуваних файлів
- Використані інструменти та їх версії
- Методику дослідження
- Знімки екрану структури бази
- Результати SQL-запитів
- Таблиці з витягнутими даними

## 6. Відновлення видалених даних

Використовуйте спеціалізовані інструменти:
- SQLite Forensic Explorer
- Undark (інструмент для відновлення)
